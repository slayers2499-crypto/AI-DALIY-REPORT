# 🔁 자동 개선 루프 스킬

이슈가 더 이상 발견되지 않을 때까지 전체 개선 사이클을 반복합니다.

## 트리거

```
이슈 없을 때까지 개선해줘
자동으로 계속 개선해줘
완전히 정리해줘
```

## 실행 흐름

```
┌─────────────────────────────┐
│  project-audit 실행         │
│  → 신규 이슈 발굴            │
└──────────┬──────────────────┘
           │
     신규 이슈 있음?
      ┌────┴────┐
     예          아니오
      │           │
      ▼           ▼
issue-resolver  doc-optimizer 1회 실행
doc-optimizer   → 완료 보고 후 종료
      │
      └── 다음 회차로 ↑
```

## 종료 조건

다음 중 하나라도 해당되면 루프 종료:

- project-audit에서 신규 이슈 **0건** 발견
- 동일한 이슈가 **2회 연속** 반복 발견 (자동 해결 불가 이슈만 남은 상태)
- **5회** 반복 후에도 이슈가 남아 있으면 중단하고 잔여 목록 보고

## 회차별 보고 형식

```
── 1회차 ──────────────────────────
project-audit  신규 N건
issue-resolver N건 해결 / N건 수동 필요
doc-optimizer  N건 갱신

── 2회차 ──────────────────────────
project-audit  신규 N건
...

── 완료 ────────────────────────────
총 N회 실행 | N건 자동 해결 | N건 수동 처리 필요
```

## 수동 처리 필요 이슈 안내

루프 종료 후 자동 해결 불가 이슈가 남아 있으면:
- GitHub 이슈 번호와 제목 목록 출력
- 각 이슈에 처리 방법 안내 코멘트 추가

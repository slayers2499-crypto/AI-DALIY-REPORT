# 성능 벤치마크

## 리포트 생성 성능 지표

| 항목 | 목표 | 측정 방법 |
|------|------|----------|
| 전체 생성 시간 | 60초 이내 | 실행 시작~파일 저장 완료 |
| WebSearch 응답 | 10초 이내 (키워드당) | 개별 검색 응답 시간 |
| WebFetch 응답 | 15초 이내 (기사당) | URL 패치 완료 시간 |
| HTML 파일 크기 | 20~100KB | 생성된 .html 파일 크기 |
| 수집 기사 수 | 5개 이상 | 실제 본문 파싱된 기사 수 |

## 2026-06-11 기준 측정값

| 항목 | 결과 |
|------|------|

| 섹션 수 | 6개 |

## 성능 개선 방향

- WebSearch 5개 키워드 병렬 실행 (현재 구현됨)
- WebFetch 기사 병렬 패치 (이슈 #26)
- 캐싱: 동일 URL 24시간 내 재요청 방지 (이슈 #12 연관)

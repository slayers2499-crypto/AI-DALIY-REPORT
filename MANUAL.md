# 사용자 매뉴얼

## 소개

AI 일일 트렌드 리포트 시스템은 매일 아침 최신 AI 뉴스와 트렌드를 자동으로 수집하여  
C-레벨 임원이 읽기 쉬운 HTML 리포트를 생성합니다.

## 빠른 시작

### 리포트 즉시 생성

Claude Code를 열고 다음 중 하나를 입력하세요:
```
오늘 AI 리포트 만들어줘
일일 리포트 생성해줘
```

생성된 파일 위치: `C:\Temp\AI-DALIY-REPORT\report_YYYYMMDD.html`

### 자동 스케줄 확인

- Claude Code 사이드바 → **Routines** 탭
- `daily-ai-trend-report` 상태가 **활성** 인지 확인
- 매일 오전 8시 자동 실행

## 리포트 읽는 법

### Executive Summary
하루를 시작하며 가장 먼저 읽어야 할 3줄 요약.  
"오늘 AI 업계에서 가장 중요한 것은 무엇인가"에 대한 답.

### 주요 뉴스
5~8건의 헤드라인과 간략한 요약.  
클릭하면 원문 기사로 이동합니다.

### 트렌드 분석
단기 뉴스를 넘어선 3~4개의 거시적 흐름.  
"이 방향이 어디로 가고 있는가"를 설명합니다.

### 전략적 시사점
C-레벨이 바로 액션을 취할 수 있는 5가지 구체적 제안.  
"우리 회사는 어떻게 대응해야 하는가"에 대한 답.

## 자주 쓰는 명령

| 상황 | 명령 |
|------|------|
| 오늘 리포트 생성 | `오늘 AI 리포트 만들어줘` |
| 리포트 품질 확인 | `리포트 분석해줘` |
| 리포트 백업 | `리포트 백업해줘` |
| 이전 리포트 복구 | `2026-06-10 리포트 복구해줘` |
| 프로젝트 상태 점검 | `프로젝트 점검해줘` |
| 이슈 자동 해결 | `이슈들 해결해줘` |

## 설정 변경

`config.json` 파일을 수정하여 커스터마이징:

### 검색 키워드 변경
```json
"search_keywords": [
  "AI news today 2026",
  "원하는 키워드 추가"
]
```

### 저장 경로 변경
```json
"storage_path": "C:\\원하는\\경로"
```

### 신뢰 출처 추가
```json
"trusted_sources": [
  "techcrunch.com",
  "추가할도메인.com"
]
```

## 문제 해결

### 리포트가 오늘 안 만들어졌어요
1. `오늘 AI 리포트 만들어줘`로 수동 생성
2. 인터넷 연결 확인
3. Claude Code 로그인 상태 확인

### 리포트 내용이 오래된 것 같아요
- `config.json`의 `max_article_age_days` 값 확인 (기본: 7일)
- `리포트 분석해줘`로 최신성 점수 확인

### 스케줄이 8시에 안 실행돼요
- Claude Code 앱이 백그라운드에서 실행 중인지 확인
- Routines 패널에서 작업 활성화 상태 확인
